# zsh

My zsh configuration, with plugins builtin via submodules. Fully independent of oh-my-zsh.

## Installation
Preferred method is to use my dotfiles repository, which will clone this repository as a submodule.
The submodule can then my symlinked to `~/.config/zsh`

Alternatively, for standalone, you can just clone this directly to `~/.config/zsh`.

## CLI Dependencies
- zsh
- fzf
- bat
- ripgrep
- eza
- fastfetch
- kitty (for png + ssh alias to kitten ssh)
